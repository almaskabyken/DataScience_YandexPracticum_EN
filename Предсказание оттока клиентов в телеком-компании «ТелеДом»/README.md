# Предсказание оттока клиентов в телеком-компании «ТелеДом»

## Описание проекта
Компания «ТелеДом» хочет снизить отток клиентов, предлагая персонализированные скидки и специальные условия пользователям, которые собираются расторгнуть договор. Для этого требуется модель машинного обучения, которая сможет предсказывать уход клиента на основе его данных и истории использования услуг.

## Данные
**Для обучения модели предоставлены 4 таблицы:**

* contract_new.csv – информация о договоре (начало, окончание, тип оплаты, расходы).
* personal_new.csv – персональные данные клиентов (пол, возраст, семейное положение).
* internet_new.csv – данные об интернет-услугах (тип подключения, дополнительные сервисы).
* phone_new.csv – информация об услугах телефонии (многолинейность).

## Этапы работы
1. Предобработка данных
2. Анализ данных
3. Подготовка данных
  * Кодирование категориальных признаков.
  * Масштабирование числовых признаков (StandardScaler).
  * Разделение данных на обучающую (75%) и тестовую (25%) выборку.
  * Балансировка классов с помощью SMOTE.
4. Обучение моделей
* SVM (метод опорных векторов).
* Random Forest (случайный лес).
* Gradient Boosting (градиентный бустинг).
5. Подбор гиперпараметров
* Используется GridSearchCV (5-кратная кросс-валидация).
* Оценивается метрика AUC-ROC (целевое значение ≥ 0.85).
5. Выбор лучшей модели
* Оценка на тестовой выборке, анализ важности признаков.

##Анализ факторов оттока:
**Наиболее значимые факторы оттока:**
* Длительность контракта: Долгосрочные контракты связаны с повышенным оттоком.
* Метод оплаты: Клиенты с автоматической оплатой чаще уходят.
* Тип интернет-сервиса: Отток выше среди пользователей оптоволоконного интернета (Fiber optic).
* Электронные счета: Использование электронного биллинга увеличивает вероятность ухода.
* Социальные факторы: Пожилые клиенты и клиенты без партнера также более склонны к оттоку.

# Рекомендации:

* Предложить гибкие тарифные планы с помесячной оплатой, поскольку клиенты с долгосрочными контрактами чаще уходят.
* Пересмотреть стратегию автоматической оплаты и предложить более понятные уведомления о платежах.
* Оптимизировать услуги на основе обратной связи клиентов с оптоволоконным интернетом.
* Уделить внимание пожилым клиентам и предложить персонализированные услуги для повышения лояльности.

**Заключение:** Построенная модель на основе CatBoost позволяет эффективно предсказывать вероятность оттока клиентов и выделяет ключевые факторы, влияющие на принятие решения об уходе. Применение предложенных рекомендаций позволит снизить процент оттока и повысить удовлетворенность клиентов.
